# xPoll

xPoll is a .NET MAUI Application designed to function as a Modbus Poll utility supporting RTU, ASCII, and TCP protocols. Built on .NET 10, it is developed to be multi-platform, ensuring compatibility across various operating systems.

## Features

- **Protocol Support**: Supports Modbus RTU, ASCII, and TCP.
- **Tag Labeling**: Allows users to set labels for each tag.
- **Data Display**: Visualizes data in formats such as:
  - Word (Signed, Unsigned)
  - 32-bit (Signed, Unsigned)
- **Swapping Support**: Implements Word and Byte Swapping functionality.

## Technology Stack

- Framework: .NET 10
- UI Framework: .NET MAUI for Multi-platform support

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/twanchai/xPoll.git
   ```
2. Navigate into the project directory:
   ```bash
   cd xPoll
   ```
3. Run the application:
   ```bash
   dotnet build
   dotnet run
   ```
